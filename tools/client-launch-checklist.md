# WebKreatives — Client Launch Checklist

Use this checklist for every new client website. Go through it top to bottom before handing over the project.

---

## 1. DOMAIN & HOSTING

- [ ] Domain registered (Hostinger, GoDaddy, Namecheap, etc.)
- [ ] Hosting plan chosen:
  - Free: GitHub Pages (static sites only)
  - Paid: Hostinger (~€3/mo), Netlify, Vercel
- [ ] Website files uploaded / GitHub repo connected
- [ ] Site live and accessible on the domain
- [ ] HTTPS / SSL certificate active (padlock visible in browser)

---

## 2. DNS RECORDS

Set these in the domain's DNS settings (Hostinger → Domains → DNS/Nameservers):

| A | @ | 185.199.108.153 | 3600 |
| A | @ | 185.199.109.153 | 3600 |
| A | @ | 185.199.110.153 | 3600 |
| A | @ | 185.199.111.153 | 3600 |
| CNAME | www | csabavk.github.io | 3600 |

### Email records (required for every client):
| Type | Name | Value | TTL |
|------|------|-------|-----|
| TXT | @ | `v=spf1 include:_spf.mail.hostinger.com ~all` | 3600 |
| TXT | _dmarc | `v=DMARC1; p=none; rua=mailto:info@CLIENT.com` | 3600 |
| CNAME | hostingermail-a._domainkey | hostingermail-a.dkim.mail.hostinger.com | 300 |
| CNAME | hostingermail-b._domainkey | hostingermail-b.dkim.mail.hostinger.com | 300 |
| CNAME | hostingermail-c._domainkey | hostingermail-c.dkim.mail.hostinger.com | 300 |
| TXT | hostingermail1._domainkey | *(generated by Hostinger — copy from Custom DKIM)* | 3600 |
| MX | @ | mx1.hostinger.com (priority 5) | 14400 |
| MX | @ | mx2.hostinger.com (priority 10) | 14400 |

> **Note:** SPF/DKIM/DMARC values may differ if client uses a different email provider (Gmail Workspace, Outlook 365, etc.). Always use the provider's own values.

---

## 3. EMAIL SETUP

- [ ] Email mailbox created (e.g. info@clientdomain.com)
- [ ] DKIM enabled in Hostinger → Emails → Custom DKIM → Verified ✅
- [ ] SPF record added (see DNS table above)
- [ ] DMARC record added (see DNS table above)
- [ ] Verify all three at [mxtoolbox.com](https://mxtoolbox.com):
  - SPF: enter domain → SPF Lookup
  - DKIM: enter `clientdomain.com:hostingermail1` → DKIM Lookup
  - DMARC: enter domain → DMARC Lookup
- [ ] Gravatar set up with client email → logo/photo uploaded
- [ ] Email signature created and installed (Hostinger webmail + Gmail)
- [ ] Auto-reply template set up (if client wants it)
- [ ] Gmail filter configured (if forwarding to Gmail)

---

## 4. CONTACT FORM

- [ ] Web3Forms account created at [web3forms.com](https://web3forms.com)
- [ ] Access key generated for client email
- [ ] Access key added to index.html hidden field
- [ ] Form tested — submission received in inbox ✅
- [ ] Captcha configured in Web3Forms dashboard (set to hCaptcha or built-in filter)
- [ ] Auto-reply template wired up to Web3Forms key

---

## 5. SEO & DISCOVERABILITY

- [ ] Page title set correctly (browser tab)
- [ ] Meta description added
- [ ] OG image created and added (for social sharing previews)
- [ ] Favicon added (favicon.svg or favicon.ico)
- [ ] sitemap.xml created and uploaded
- [ ] robots.txt created and uploaded
- [ ] Google Search Console:
  - [ ] Account created / client added
  - [ ] Meta verification tag added to index.html
  - [ ] Sitemap submitted
- [ ] Google Analytics:
  - [ ] GA4 property created
  - [ ] Tracking script added to index.html
- [ ] Google Business Profile:
  - [ ] Created / claimed
  - [ ] Category set
  - [ ] Short description added
  - [ ] Website URL added
  - [ ] Photos uploaded (logo, cover, team)
  - [ ] Service area set (if no physical address)

---

## 6. GITHUB (version control)

- [ ] Repo created (public for GitHub Pages)
- [ ] All files committed and pushed
- [ ] GitHub Pages enabled → Source: main branch / root
- [ ] Custom domain added in Pages settings
- [ ] DNS check shows green ✅
- [ ] Enforce HTTPS checked ✅
- [ ] README.md updated with project description

---

## 7. HANDOVER TO CLIENT

- [ ] Client preview sent (client-preview.html with watermark)
- [ ] Client approved design ✅
- [ ] Invoice sent
- [ ] Login credentials handed over:
  - [ ] Hostinger account / email login
  - [ ] Webflow editor login (if applicable)
  - [ ] Google Analytics access
  - [ ] Google Search Console access
  - [ ] Google Business Profile ownership transferred
- [ ] Short walkthrough done (video call or Loom recording)
- [ ] Client added to monthly plan (if subscribed)
