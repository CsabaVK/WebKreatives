# WebKreatives

Web design agency website for small businesses. Built with HTML, CSS, and JavaScript.

## Structure

```
webkreatives/
├── index.html          # Main page
├── css/
│   └── style.css       # All styles
├── js/
│   └── script.js       # Scroll animations
└── assets/
    └── logo.png        # Logo (export from Canva)
```

## Sections

- Hero with browser mockup
- Trust bar & stats
- Services (6 cards)
- Why Us (dark section)
- Project showcases (3 case studies)
- Testimonials
- Pricing (3 tiers: €995 / €2,495 / €4,950)
- Contact form
- Footer
